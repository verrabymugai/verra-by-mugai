How to setup Google Sheets ▼
  1. Create a new Google Sheet.
  2. Configure column headers in row 1:
    Timestamp, Name, Phone, Email, Room, Check-in, Check-out, Guests, Notes
  3. Go to Extensions > Apps Script.
  4. Delete default script code and paste this:
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date(), data.name, data.phone, data.email, data.room, data.checkin, data.checkout, data.guests, data.comments || "" ]); var response = { result: 'success', message: 'Row appended successfully' }; return ContentService.createTextOutput(JSON.stringify(response)) .setMimeType(ContentService.MimeType.JSON) .addHeader('Access-Control-Allow-Origin', '*'); }
  1. Click the save icon, then click Deploy > New Deployment.
  2. Choose Web App as deployment type.
  3. Set "Execute as" to Me.
  4. Set "Who has access" to Anyone.
  5. Click Deploy, authorize permissions, and copy the generated Web App URL.
  6. Paste the URL in the input box above and click Save!
